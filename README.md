IPL Dashboard 🏏
A simple and interactive React application to track IPL teams and their match statistics.

What it does?
Team List: Shows all IPL teams on the home page.

Match Details: Displays the latest match info and recent match history for each team.

Statistics: Shows a Pie Chart (Wins vs Losses) for each team using Recharts.

Responsive: Changes theme colors based on the selected team.

Tech Used
Frontend: React JS

Routing: React Router

Charts: Recharts

API: CCBP IPL API

How to use?
Open the app.

Click on any IPL Team card.

View the match stats and the performance chart.

Click the Back button to return home.
